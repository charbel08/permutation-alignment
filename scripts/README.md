# Scripts Layout

This folder is grouped by purpose:

- `ablation/`: ablation experiments and launchers
- `eval/`: evaluation scripts (memorization, MMLU/Qwen)
- `keys/`: key generation/validation utilities
- `data/`: data prep/transformation scripts
- `mila/`: Mila cluster launchers
- `snow/`: Snow cluster launchers

Cluster-specific workflows remain under `mila/` and `snow/`.
